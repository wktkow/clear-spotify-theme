## Installation (Desktop App)

### On Linux:

1. Make sure you have [spicetify](https://spicetify.app/) installed.

2. Run the installer:
   ```bash
   curl -fsSL https://raw.githubusercontent.com/wktkow/clear-spotify-theme/main/sh/install.sh | bash
   ```
   (Kills Spotify, clears any previous theme/snippets, downloads theme files, applies, and launches Spotify)

3. Enjoy!

### On Windows:

1. Open Powershell as administrator

2. Copy and paste the below code snippet:

```powershell
iwr -useb https://raw.githubusercontent.com/spicetify/cli/main/install.ps1 | iex
```

(Installs [spicetify](https://spicetify.app/))

3.  Copy and paste the below code snippet:

```powershell
iwr -useb https://raw.githubusercontent.com/wktkow/clear-spotify-theme/main/powershell/install.ps1 | iex
```

(Installs [Clear Spotify](https://github.com/wktkow/clear-spotify-theme) â€” kills Spotify, clears any previous theme/snippets, downloads theme files, applies, and launches Spotify)

4. Enjoy!
